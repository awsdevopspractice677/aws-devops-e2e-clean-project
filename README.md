# AWS DevOps End-to-End Clean Project

This project demonstrates a complete DevOps CI/CD pipeline built from scratch on AWS using clean, production-grade practices.

## Tools & Technologies
- Git & GitHub
- Maven
- Jenkins
- Docker
- Kubernetes (EKS)
- Terraform
- Ansible
- Prometheus & Grafana
- AWS

## Objective
To design, build, and deploy a scalable application with full CI/CD automation and monitoring.
